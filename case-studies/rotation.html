Rotation – A Music Feed for Record Stores

Turning static Discogs listings into swipeable, playable discovery feeds

  • Learn More

Overview

Rotation is a semantic discovery tool built for vinyl lovers. It connects to Discogs and transforms static record listings into a scrollable, audio-rich feed — so DJs, collectors, and casual diggers can preview and wishlist records the way they explore in real life: by listening.

Key Achievements

  • Designed and built a prototype using Next.js, Discogs OAuth, and YouTube audio matching
  • Developed wishlist functionality, semantic tags, and swipe-based UI for intuitive discovery
  • Defined user roles, store upload flow, and storefront management for record sellers

Challenge

Discogs is functional but not discoverable. It's built for collectors, not listeners. Most listings have no audio, chaotic embeds, and flat layouts — making it hard for stores to showcase and harder for buyers to explore.

Insight

Most record stores don't need a new store — they need a better way to show off what they already have. If you can turn Discogs into a scrollable feed with playable previews, you create a bridge between browsing and listening.

Action

Built Rotation to pull store inventories from Discogs, match track previews from YouTube and Apple Music, and display them in a mobile-first feed with swipe gestures. Added wishlist and buy link integration, with a roadmap toward personalisation and pricing analytics.

Results

Currently in early testing. Demos with DJs and store owners have validated strong interest, especially for high-volume stores struggling to differentiate. Positioned as a lightweight tool that adds serious value without disrupting existing workflows.
